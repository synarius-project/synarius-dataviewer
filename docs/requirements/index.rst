Requirements
============

ID policy matches the other Synarius repositories:

- ``<REPO>-<LEVEL>-NNN`` for system and architecture requirements
- ``<REPO>-COMP-<COMP>-NNN`` for component requirements

For this repository, use ``DV`` as the repo code (for example ``DV-SYS-001``).

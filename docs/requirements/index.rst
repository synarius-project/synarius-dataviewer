Requirements
============

ID policy matches the other Synarius repositories:

- ``<REPO>-<LEVEL>-NNN`` for system and architecture requirements
- ``<REPO>-COMP-<COMP>-NNN`` for component requirements

For this repository, use ``DV`` as the requirement ID prefix (for example ``DV-SYS-001``).
The Git / packaging name is **synarius-apps**; IDs need not change when the repo was renamed.
